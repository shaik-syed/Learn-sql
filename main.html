Introduction to SQL

SQL (Structured Query Language) is the standard language used for managing and manipulating relational databases.

Basic SQL Commands

SELECT

Used to query data from a database.

            SELECT column1, column2 FROM table_name;

INSERT

Used to insert data into a table.

            INSERT INTO table_name (column1, column2) VALUES (value1, value2);

SQL Joins

INNER JOIN

Returns records with matching values in both tables.

            SELECT * FROM employees INNER JOIN departments ON employees.dept_id = departments.dept_id;

SQL Subqueries

Definition

A subquery is a query inside another query.

            SELECT name FROM employees WHERE dept_id = (SELECT dept_id FROM departments WHERE location = 'New York');

SQL Constraints

PRIMARY KEY

Ensures that each record in a table is unique.

            CREATE TABLE employees (
                emp_id INT PRIMARY KEY,
                name VARCHAR(100)
            );

SQL Indexing

Indexing improves the speed of data retrieval.

            CREATE INDEX idx_name ON employees(name);

SQL Views

A view is a virtual table based on the result of a query.

            CREATE VIEW employee_view AS
            SELECT * FROM employees WHERE age > 30;

SQL Transactions

COMMIT

Saves the current transaction.

            COMMIT;

ROLLBACK

Rolls back the transaction to the previous state.

            ROLLBACK;

SQL Grouping & Aggregation

COUNT

Returns the number of rows.

            SELECT COUNT(*) FROM employees;

SUM

Returns the sum of values.

            SELECT SUM(salary) FROM employees;

SQL Sorting

Used to sort the result set.

            SELECT * FROM employees ORDER BY salary DESC;

SQL Wildcards

Used for pattern matching.

            SELECT * FROM employees WHERE name LIKE 'A%';

SQL Functions

NOW()

Returns the current date and time.

            SELECT NOW();

CONCAT()

Combines two or more strings into one.

            SELECT CONCAT(first_name, ' ', last_name) AS full_name FROM employees;

SQL Normalization

Normalization is the process of organizing data to avoid redundancy and improve data integrity.

SQL Data Types

  • INT: Stores integer values
  • VARCHAR: Stores variable-length strings
  • DATE: Stores date values
